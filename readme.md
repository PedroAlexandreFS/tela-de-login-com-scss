# Fake Login Page

Este projeto é uma página de login fake desenvolvida em **HTML** e **SCSS**, com foco em estilização e boas práticas de organização de código.

## 📂 Estrutura do Projeto

├── index.html        # Estrutura principal da página
├── style/
│   ├── index.css     # CSS compilado
│   ├── variaveis.scss # Variáveis de cores e fontes
│   ├── mixin.scss     # Mixins reutilizáveis
│   ├── butoes.scss    # Estilo dos botões

## 📸 Desmosntração
Tela de login simples e estilizada com SCSS.
![tela login](/img/Captura%20de%20tela%202026-05-04%20165926.png)

## 🚀 Tecnologias Utilizadas

- **HTML5**: Estrutura semântica da página.
- **SCSS (Sass)**: Pré-processador CSS para melhor organização e reutilização.
  - Uso de variáveis (`$blue`, `$red`, `$white`, etc.).
  - Uso de mixins (`flex-centralizado`, `input-style`).
  - Uso de `@use` para modularização.
- **CSS3**: Estilização final compilada.

## 🎨 Funcionalidades

- Formulário de login com campos de **Email** e **Senha**.
- Checkbox "Lembra-me".
- Botão estilizado com efeitos de hover.
- Ícones nos inputs (envelope e cadeado).
- Layout centralizado com **flexbox**.
- Fundo com **gradiente linear**.
